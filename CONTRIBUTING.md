# Contributing

Thanks for helping improve Maintainer Compass.

## Development

```bash
python -m pip install -e .
python -m unittest discover -s tests
```

## Pull Requests

- Keep changes focused.
- Add or update tests for behavior changes.
- Update README or docs when user-facing commands change.
- Include before-and-after output when changing reports.

## Good First Issues

Useful starter tasks include adding new repository checks, improving report wording, and adding examples for common ecosystems.
